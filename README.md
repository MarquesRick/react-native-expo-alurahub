<p align="center">🚀 simple application developed with React Native CLI using Json Server for CRUD repositories</p>

###  🏗  Running/Creating the project
```bash
# run json server 
json-server --watch --host 192.168.0.101 db.json

# run expo on IOS
expo start --ios

# run expo on Android
expo start --android
```
### 💿 Technologies

- [Node.js](https://nodejs.org/en/)
- [React Native](https://reactnative.dev/)
- [React Hooks](https://pt-br.reactjs.org/docs/hooks-intro.html)
- [React Navigation](https://reactnavigation.org/)
